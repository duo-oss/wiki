# babel
## Babel 是一个 JavaScript 编译器
Babel 通过语法转换器来支持新版本的 JavaScript 语法。
Babel 能够转换 JSX 语法
`@babel/preset-env`  是一个插件集合 目的是不用你一个一个把插件引入进来， 只要引入这个 就可以了
